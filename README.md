# 🌾 AgriGenius AI – Smart Farmer Platform

An AI-powered farming platform for Indian farmers built with React + Claude AI.

## 🚀 Features

- 🔬 AI Soil Scanner
- 🛰️ Satellite Farm Monitoring
- 🌾 Crop Recommendation Engine
- 💹 Real-Time Mandi Prices
- 🧪 Fertilizer Price Tracker
- ⛅ Weather Intelligence
- 🐛 Pest & Disease Detection
- 💰 Farm Profit Analyzer
- 🏛️ Government Scheme Hub
- 🎤 Voice Assistant (Kannada/Hindi/English)
- 🛒 Farm Marketplace

## 🌐 Live Demo
[View Live App](https://agri-genius-ai.vercel.app)

## 🛠️ Tech Stack
- React + Vite
- Claude AI (Anthropic)
- Tailwind CSS

## 📱 Built for Indian Farmers
Supports Kannada, Hindi and English languages.
